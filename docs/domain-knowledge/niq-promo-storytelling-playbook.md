# NIQ Promo Storytelling Playbook

This playbook defines how Basquio should analyze NielsenIQ-style promotion data.

It is not enough to mention promotions. A promo deck must show the analytical mechanics that explain why promotional performance changed.

## Core Principle

Use SCQA only as the wrapper.

The analytical engine must be a drill-down across:

- market / segment
- channel
- area / retailer
- brand / competitor
- format / pack
- promo vs no-promo
- discount tier
- WD Promo / display / folder
- rotation / productivity

The target quality bar is at least as deep as a strong NielsenIQ promo review and ideally better.

## Mandatory Promo Cascade

For promotion-led briefs, follow this sequence unless the input genuinely lacks the required data.

1. Category baseline
   - sales value
   - sales volume
   - average price
   - promo sales vs no-promo sales
2. Inflation / price tension test
   - if value and volume diverge, show average price and say whether value is inflated by pricing
   - when price inflation materially distorts value, pivot the commercial story to volume
3. Promo pressure
   - promo intensity
   - baseline / incremental if available
   - discount tier decomposition
4. Competitive read
   - focal brand versus the main competitor(s)
   - the focal brand must remain visible even when small
5. Localization
   - channel
   - retailer / area
   - format / pack
6. Promo mechanics
   - WD Promo
   - display
   - folder
   - depth of discount
   - timing by month if available
7. Opportunity statement
   - only after the deck has shown the gap and the mechanism
8. Final synthesis
   - 2-4 actions max in short / medium decks

## Non-Negotiable Intelligence Rules

### 1. Value-first is not always the commercial truth

If prices rose materially, show value, show price, then pivot to volume.

Bad:
- "Macinato is the strongest growth pool" when value is inflated by price and volume is weak

Good:
- "Macinato grows in value, but the gain is price-led; volume is weaker than the headline suggests"

### 2. The chart must prove the claim

If the slide says:

- rotation problem
- productivity gap
- price-led growth
- distribution opportunity

then the hero exhibit must show:

- rotation / ROS / productivity
- price mechanics or value-vs-volume decomposition
- distribution plus productivity proof

Never hide the real evidence in a side note while charting a generic KPI.

### 3. Opportunity claims need mechanism proof

Do not say "there is a distribution opportunity" unless the deck shows at least one of:

- rotation / ROS
- value per distribution point
- fair-share index
- WD Promo gap plus promo productivity proof

### 4. Focal brand persistence

Whenever the deck discusses:

- players
- suppliers
- competitors
- manufacturer rankings

the focal brand must still be located explicitly on the slide.

### 5. No invented motives

Do not assign strategic intent or brand positioning unless the data shows it.

Bad:
- "Borbone grows because it is a premium Italian brand"

Good:
- "Borbone grows in capsules because it gains WD Promo and promo volumes in the fastest-growing format"

### 6. Kill redundant analytical cuts

If a later slide uses the same analytical cut as an earlier slide, it must introduce at least one of:

- deeper grain
- different metric family
- different causal driver
- different decision-relevant comparison set

Different prose on the same chart logic is not a new slide.

### 7. Keep branches contiguous

Do not jump from one branch to another and then back again.

Bad:

- category
- segments
- channels
- segments again

Good:

- category
- segments
- channels / retailers
- promo mechanics
- synthesis

Only revisit an earlier branch if it is explicitly a synthesis/comparison or a clearly deeper follow-up.

## Exhibit Selection Rules For Promo Work

### Use these patterns

- Category baseline: grouped bars for value vs volume + line for average price or separate mini-panels
- Promo vs no-promo: split bars / stacked bars / small multiples
- Discount tiers: grouped or stacked bars by price reduction band
- Timing mechanics: monthly bars or lines
- Focal brand vs competitor: paired panels plus evidence table
- Distribution opportunity: distribution vs productivity scatter or table with WD / ROS / sales per point

### Avoid these anti-patterns

- generic market-share ranking when the real issue is price
- sales chart when the slide claim is rotation
- recommendation slide introducing evidence not shown before
- repeating a market overview chart and only changing the commentary

## Recommendation Compression Rule

Promo decks should usually end with a short synthesis, not a theater of recommendation cards.

For short and medium decks:

- 1-2 recommendation slides
- each action must compress prior evidence
- no new unsupported claim should first appear there

## Working Standard

Basquio should preserve client-friendly copy, but friendliness is never allowed to overwrite:

- evidence traceability
- metric / exhibit coherence
- focal-brand clarity
- inflation-aware interpretation
- productivity-backed opportunity logic
